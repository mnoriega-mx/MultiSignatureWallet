# MultiSignatureWallet

Programmed a Multisig wallet, that consists of a smart contract that allows the contract holder to set multiple addresses as owners of the wallet, which will be able to withdraw ether as long as their transaction reaches consensus. Consensus is reached when the transaction requests reach the approval limit, set by the contract holder. Every contract owner can contribute to the approval of a transaction. This project was fully programmed in Solidity.
